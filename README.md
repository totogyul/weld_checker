# 용접 오매칭 검사기

통합문서 PDF(용접 로그)와 도면 PDF를 비교해 오매칭 항목을 Excel로 출력하는 도구입니다.

## 사용 방법

### 실행 파일 (Windows)

1. [Releases](../../releases) 또는 [Actions](../../actions) 탭에서 `용접오매칭검사기.exe` 다운로드
2. 더블클릭으로 실행 → 브라우저 자동으로 열림
3. 통합문서 PDF, 도면 PDF 업로드
4. **검사 후 Excel 다운로드** 클릭 → `오매칭_검사결과.xlsx` 저장

> Python 설치 불필요 / 인터넷 연결 불필요 / 데이터 외부 전송 없음

### 결과 Excel 구성

| 시트 | 내용 |
|------|------|
| 도면에만있음_로그누락 | 도면엔 있으나 통합문서에 없는 쌍 |
| 로그에만있음_도면누락 | 통합문서엔 있으나 도면에 없는 쌍 |
| 정상매칭 | 양쪽 모두 일치하는 쌍 |
| 도면_미매칭항목 | 도면에서 짝을 찾지 못한 항목 (파싱 이상 확인용) |
| 요약 | 전체 건수 요약 |

## 코드 규칙

- **용접사 코드**: 알파벳 2자리 + 숫자 (예: `HM-001`, `HG-02`)
- **용접번호**: `W-` + 숫자 + 선택적 알파벳 (예: `W-01`, `W-04R`)
- 앞 0은 무시하고 비교 — `HM-004 = HM-04 = HM-4` / `HM-0040 ≠ HM-04`

## 개발 환경에서 실행

```bash
pip install -r requirements.txt
python app.py
```

## 기술 스택

- Python / Flask
- pdfplumber (PDF 파싱)
